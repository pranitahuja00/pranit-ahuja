Wikipedia Edit Anomaly Detection Pipeline
Real-time streaming pipeline that ingests live Wikipedia edits from the Wikimedia SSE API via Kafka into Databricks Structured Streaming. Bronze-Silver-Gold medallion architecture with feature engineering in the silver layer (revert detection, byte delta, namespace mapping) and rolling Z-score anomaly flagging across 1-min and 5-min tumbling windows in the gold layer. Results surface in a live Streamlit dashboard backed by Databricks SQL.
